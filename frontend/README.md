# CRM Simples - Frontend

## Getting Started

Na raiz do projeto, inicie frontend e backend juntos:

```bash
cd /home/anselmo/crm_anselmotech
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000). A API fica em
[http://localhost:8000](http://localhost:8000) e a documentação em
[http://localhost:8000/api/v1/docs](http://localhost:8000/api/v1/docs).

Antes de iniciar, configure `frontend/.env.local` e `backend/.env` usando os
respectivos arquivos `.env.example`.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
